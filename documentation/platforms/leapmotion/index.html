Leapmotion

Repository Issues

The Leap Motion is a user-interface device that tracks the user's hand motions, and translates them into events that can control robots and physical computing hardware.

For more info about the Leap Motion platform click here.

How to install

Installing Cylon.js with leapmotion support is pretty easy.

npm install cylon-leapmotion

How to use

This small program uses a Leap Motion to detect the user's hand position

JavaScript

Cylon = require('cylon');

Cylon.robot({
  connection: {
    name: 'leapmotion',
    adaptor: 'leapmotion',
    port: '127.0.0.1:6437'
  },
  device: {
    name: 'leapmotion',
    driver: 'leapmotion'
  },
  work: function(my) {
    return my.leapmotion.on('hand', function(hand) {
      return Logger.info("" + hand.palmX + "," + hand.palmY + "," + hand.palmZ);
    });
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection: { name: 'leapmotion', adaptor: 'leapmotion', port: '127.0.0.1:6437' }
  device: { name: 'leapmotion', driver: 'leapmotion' }

  work: (my) ->
    my.leapmotion.on 'hand', (hand) ->
      Logger.info "#{hand.palmX},#{hand.palmY},#{hand.palmZ}"

.start()

How to connect

OSX

This driver works out of the box with the vanilla installation of the Leap Motion Software that you get in their Setup Guide.

The main steps are:

  • Run Leap Motion.app
  • Connect your Computer and Leap Motion Controller.
  • Connect to the device via Cylon.js.

Ubuntu

The Linux download of the Leap Motion software can be obtained from Leap Motion Dev Center (requires free signup)

The main steps are:

  • Run the leapd daemon.
  • Connect your computer and the Leap Motion controller
  • Connect to the device via Cylon.js
