What is Cylon.js?

Cylon.js is a JavaScript framework for robotics and physical computing using Node.js.

It provides a simple, yet powerful way to create solutions that incorporate multiple, different hardware devices at the same time.

How to Install Node.js

OSX

Go to the Node.js downloads page at http://nodejs.org/download/ download the installer file and run it.

Ubuntu

Follow the installation instructions at https://github.com/joyent/node/wiki/Installation to install the latest Node.js

Windows

Go to the Node.js downloads page at http://nodejs.org/download/ download the installer file and run it.

How to Install CoffeeScript (Optional)

If you want to program Cylon.js using CoffeeScript, it is easy to install using NPM, the Node Package Manager: 

npm install -g coffee-script

You might need to install under the 'root' user by running the above install command using `sudo`.

How to Install Cylon.js

In order to use Cylon.js, you must install the Cylon.js node modules needed to control your particular hardware setup. For example, to use Cylon.js with an Arduino, you will need to install the `cylon-firmata` module. Check out this example: 

npm install cylon
npm install cylon-firmata

Cylon.js Robots

Writing your robot code:

Now you are ready to write your own code. Take a look at the examples below, or go to the examples directory for a whole bunch of code you can use to help get started. We recommend using Test-Driven Robotics (TDR) with your preferred test frameworks.

Running your robot:

node robotfilename.js

or

coffee robotfilename.coffee

Cylon.js Code Examples

Example: Hello

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: { name: 'looped', adaptor: 'loopback'},

  work: function() {
    every((1).second(), function() { Logger.info("Hello, human!"); });
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  work: ->
    every 1.second(), ->
      Logger.info("Hello, human!")

.start()

Example: Arduino with LED and a button

To use this example, you must first run npm install cylon-firmata

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: { name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0' },
  devices: [{name: 'led', driver: 'led', pin: 13},
            {name: 'button', driver: 'button', pin: 2}],

  work: function(my) {
    my.button.on('push', function() {my.led.toggle()});
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  devices:
    [
      {name: 'led', driver: 'led', pin: 13},
      {name: 'button', driver: 'button', pin: 2}
    ]

  work: (my) ->
    my.button.on 'push', -> my.led.toggle()

.start()

Parrot ARDrone 2.0

To use this example, you must first run npm install cylon-ardrone

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'ardrone', adaptor: 'ardrone', port: '192.168.1.1'},
  device: {name: 'drone', driver: 'ardrone'},
  work: function(my) {
    my.drone.takeoff();

    after((10).seconds(), function() {
      my.drone.land();
    }

    after((15).seconds(), function() {
      my.drone.stop();
    }
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection:
    name: 'ardrone', adaptor: 'ardrone', port: '192.168.1.1'

  device:
    name: 'drone', driver: 'ardrone'

  work: (my) ->
    my.drone.takeoff()

    after 10.seconds(), ->
      my.drone.land()

    after 15.seconds(), ->
      my.drone.stop()

.start()

CLI

Cylon has a Command Line Interface (CLI) so you can access important features right from the command line.

Usage: cylon [command] [options]

Commands:

  generate <name>        Generates a new adaptor

Options:

  -h, --help     output usage information
  -V, --version  output the version number

Generator

Want to integrate a hardware device we don't have Cylon support for yet? There's a generator for that!

You can easily generate a new skeleton Cylon adaptor to help you get started. Simply run the cylon generate command, and the generator will create a new directory with all of the files in place for your new adaptor module.

$ cylon generate awesome_device
Creating cylon-awesome_device adaptor.
Compiling templates.

$ ls ./cylon-awesome_device
